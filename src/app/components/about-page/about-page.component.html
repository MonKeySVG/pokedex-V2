About OnboardEx

Welcome to OnboardEx, your ultimate Pokémon browsing experience! This project is designed to provide a comprehensive and interactive way to explore the world of Pokémon. Below, you'll find details about the technologies we used, the features we implemented, and some behind-the-scenes insights.

Technologies Used

  • Angular: Angular is the framework that powers our application, providing a robust structure and efficient data binding.
  • TypeScript: We used TypeScript to ensure type safety and improve code quality.
  • NPM: NPM is used for managing dependencies and scripts.

Features

  • Filter Module: Our filter module allows you to filter Pokémon by type, generation, status, and name. This makes it easy to find exactly what you're looking for.
  • Pokedex: The Pokedex component displays a list of Pokémon with their details, including types, stats, and images.
  • Pokemon Details: Clicking on a Pokémon card takes you to a detailed page with comprehensive information about that Pokémon, including its Japanese name, sensitivities, and more.
  • Responsive Design: The application is fully responsive, ensuring a great user experience on both desktop and mobile devices.

Behind the Scenes

Our application leverages the PokeAPI to fetch detailed information about each Pokémon. We use a combination of Angular services and components to manage data and state efficiently. Here are some interesting details you might not notice at first glance:

  • Responsive Layout: The layout adjusts dynamically based on the screen size, ensuring a consistent user experience across different devices.
  • Type Sensitivities: The sensitivities of each Pokémon are calculated based on their types, providing accurate information about their strengths and weaknesses.
  • Local Storage: To improve performance, we cache Pokémon data in local storage, reducing the need for repeated API calls.
  • Shiny Probability: There is a 10% probability that a Pokémon will be displayed as shiny, adding an element of surprise and excitement for users.

Conclusion

We hope you enjoy using OnboardEx as much as we enjoyed building it. Our goal is to provide a seamless and enjoyable experience for all Pokémon enthusiasts. If you have any feedback or suggestions, feel free to reach out to us!
